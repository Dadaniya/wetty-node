# wetty-node

## 1.一个类似 wetty功能的服务
最近一个项目中需要SSH进去执行一些命令，所以就有了这个小程序，兼容windows linux,IE,chrome，主要知识点：

-------------

> * nodejs的进程模块 
> * nodej简单的web服务 
> * 简单的 AJAX发起请求 

## 2.运行程序前请自主下载依赖：

* **iconv（for unwindows）**       
* **iconv-lite(for windows)**

## 3.使用方法
* 启动start.js
* 浏览器中访问localhost:8844即可
